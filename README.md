## Setup

Follow these steps to set up the project locally on your machine:

### Prerequisites

- Node.js (v14.x or later)
- npm (v6.x or later)

### Clone the Repository

- Fork the Repo!

```bash
git clone https://github.com/<your Username>/Shopy.git
```

```bash
cd Shopy
```

```bash
npm install
```


```bash
npm run dev
```
